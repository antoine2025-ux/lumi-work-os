# Org Center – UX Improvements (Backlog for L10)

High-level ideas only. Each bullet should be small enough to fit into a 1–2 day task and make a noticeable difference for admins.

---

## 1. Org Overview

**Potential improvements (pick 2–3 for L10):**

- [ ] Make metrics strip clickable (e.g. clicking "People" jumps to People page with relevant filter).
- [ ] Add small trend indicators on metrics (e.g. +3 people in last 30 days) once we have the data.
- [ ] Add a "recent changes" mini-feed (e.g. "Team Growth created", "Invited ana@example.com") under the metrics.

---

## 2. People

**Potential improvements:**

- [ ] Add real filters (team, department, role) to replace placeholder filter pills.
- [ ] Add "Compact mode" toggle to shrink row height for large orgs.
- [ ] Add inline indicators for people missing key info (e.g. "No team yet", "No role yet") with quick actions.

---

## 3. Structure

**Potential improvements:**

- [ ] Add inline editing for team/department names from the table/cards.
- [ ] Show a simple "headcount by department" chart above the tabs.
- [ ] Add quick "Create team in this department" action directly on department cards.

---

## 4. Org Chart

**Potential improvements:**

- [ ] Add a zoom/fit control for complex org charts.
- [ ] Allow toggling between "by department" and "by reporting line" views (if/when reporting lines are modeled).
- [ ] Highlight the current user's position in the org chart (e.g. "You are here").

---

## 5. Activity & Exports

**Potential improvements:**

- [ ] Add filters for activity type (invites, structure changes, permissions changes).
- [ ] Add a simple "Last 7 days / 30 days" toggle.
- [ ] Allow scheduling a weekly CSV export email to Org Owners (if/when notifications infra exists).

---

## 6. Org Settings & Permissions

**Potential improvements:**

- [ ] Allow customizing certain capabilities per org (e.g. Member can manage invites in small teams).
- [ ] Add inline explanations on hover for each capability in the permissions table.
- [ ] Add a simple "Role simulation" mode to see what the Org Center looks like as each role.

---

## 7. Global Org UX

**Potential improvements:**

- [ ] Add an entry point from Loopwell's global navigation: "Org Center" (with short tooltip).
- [ ] Add "Back to Org overview" breadcrumb on all Org pages for better orientation.
- [ ] Add contextual "Learn more" links to internal docs from key pages (Overview, People, Structure, Activity).

---

# NOTES

- This backlog is intentionally non-committal: it lists **candidate** improvements, not a final roadmap.
- When planning L10, pick the top 3–5 items that:
  - Are feasible within a short UX pass.
  - Best support current customer use cases.
- Keep this document updated as feedback comes in from internal demos and early users.
