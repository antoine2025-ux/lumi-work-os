# Org Center Roadmap

This document tracks the development milestones for Org Center, a core feature of Loopwell.

---

## Milestones

### ✔ Milestone L11 – Completed  

- Cross-page navigation cohesion  
- Structure editing (teams, departments, roles)  
- Filters/sorting for People  
- UI polish  
- QA frameworks  

**Status:** Complete. See `org-center-l11-recap.md` for full summary.

---

### ✔ Milestone L12 – Completed  

**"Roles, Permissions & Member Management Enhancements"**

Focus areas:
- Full Org role model (Owner / Admin / Member)
- Capability matrix implementation
- Dynamic UI gating based on permissions
- Backend permission enforcement
- Member management and invite permissions

**Status:** Complete. See `org-center-l12-summary.md` for full details.

---

### ➜ Milestone L13 – Next  

**"Org Insights & Reporting"**

Focus areas:
- Lightweight analytics within Org Center
- Headcount and role distribution metrics
- Growth trends (join rates over time)
- Department/team breakdowns
- Simple charts and visualizations

**Status:** Planning phase. See `org-center-l13-plan.md` for details.
